র্তমানে ব্যবহৃত কম্পিউটার মাউসটি নষ্ট হয়ে যাওয়ায় স্বাভাবিকভাবে কাজ করতে সমস্যা হচ্ছে। ফলে দৈনন্দিন অফিসিয়াল কাজ ব্যাহত হচ্ছে।

অতএব, কাজের স্বাভাবিক গতি বজায় রাখার স্বার্থে একটি নতুন কম্পিউটার মাউস ক্রয়ের জন্য সদয় অনুমতি ও প্রয়োজনীয় ব্যবস্থা গ্রহণ করার জন্য বিশেষভাবে অনুরোধ জানাচ্ছি।

আপনার সদয় বিবেচনার জন্য কৃতজ্ঞ থাকব।
